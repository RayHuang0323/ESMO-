# AI 專案說明

## 文件目的

本文件提供所有 AI（ChatGPT、Claude、Codex、Gemini 或其他 AI）快速理解 ESMO 專案。

開始協助開發前，請先閱讀本文件。

---

# 專案定位

ESMO 是一款 Web 3D MOBA 遊戲。

主要目標：

* 可長期維護
* 可持續擴充
* 可多人協作（未來）
* AI 能快速理解專案

---

# 開發者背景

開發者不是專業工程師。

主要負責：

* 遊戲玩法
* 系統規劃
* UI
* 世界觀
* 英雄設計
* 技能設計

AI 主要協助：

* 程式撰寫
* 架構規劃
* Bug 修正
* 文件整理
* 效能優化

回答時請避免過度使用艱深的工程術語。

---

# 技術架構

目前使用：

* React
* Vite
* React Three Fiber（R3F）
* Three.js
* JavaScript

除非有重大理由，不建議大幅更換技術架構。

---

# AI 協作原則

回答問題時：

1. 優先維持目前架構。
2. 不要因方便而大幅重構。
3. 優先容易維護。
4. 優先容易閱讀。
5. 優先容易擴充。
6. 修改前先規劃。
