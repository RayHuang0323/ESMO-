## 目前最大問題

實測發現：首頁點 MOBA 後，可能錯誤進入單一「選手詳細 / Player Profile」畫面。

正確流程應該是：

Dashboard  
→ MOBA 5 人賽前配置  
→ 配對  
→ Ban/Pick  
→ 戰術  
→ Loading  
→ 3D Battle  
→ Result  
→ Dashboard
