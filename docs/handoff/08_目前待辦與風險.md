## 當前最高優先問題

首頁點 MOBA 後，實測可能進入單一「選手詳細 / Player Profile」。

這是流程錯接。

正確應為：

Dashboard  
→ MOBA 5 人賽前配置  
→ 配對  
→ Ban/Pick  
→ 戰術  
→ Loading  
→ Battle  
→ Result

單一選手詳細只能作為 Modal / Bottom Sheet / 詳細檢視，不能取代 5 人配置主頁。
